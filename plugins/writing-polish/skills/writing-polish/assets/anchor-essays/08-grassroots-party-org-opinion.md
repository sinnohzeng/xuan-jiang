# 范文 08：基层党组织任期意见

**来源**：《怎样写作》（任仲然）第六讲「规范性公文的写法」§ 意见的起草
**类型**：意见（党的基层组织任期）
**学什么**：意见的写法——一步入题、不议论、不铺垫

---

## 写作背景

这是贯彻落实党的十九大党章修正案关于党的基层组织任期的实施性规定。任仲然以此为例说明意见的标准写法。

## 原文（节录）

> 这个意见是贯彻落实党的十九大党章修正案关于党的基层组织任期的实施性规定。
>
> **第一条**把基层党委、党总支、党支部，以及村和社区基层党组织的任期，都一一规定清楚了。
>
> **第二条、第三条**是围绕第一条规定提出的工作要求。
>
> **第四条**是专门就军队党的基层组织任期问题作出说明。
>
> **第五条**是关于此规定的施行日期及与其他相关规定的关系。

## 任仲然的关键论断

> 此类**专项性意见**，通常**不需要议论，不用加以铺垫，一步入题即可**。
>
> 有些**综合性意见**，往往会先写一些认识，然后再作出规定或提出要求。即便是写认识也不宜多，而且一定要紧扣意见的主题写，**不能把意见写成论述性文章**。

## 解析

### 专项性 vs 综合性 — 写法不同

| 类型 | 是否要写"认识" | 结构 |
|---|---|---|
| **专项性意见**（如任期规定） | ❌ 不需要 | 直接 5 条规定，一步入题 |
| **综合性意见**（如全局工作） | ✅ 简短认识开头 | 认识 + 规定（认识不宜多） |

### "意见"的标准结构

```
第一条：核心规定（最具体、最操作化）
第二、三条：围绕第一条的工作要求
第四条：特殊情形说明（如有）
第五条：施行日期 + 与其他规定的关系（标准结尾）
```

### 语言特点

- **每条都给具体规定**：不"应当 / 应该 / 适当"，而是"基层党委 / 党总支 / 党支部 / 村和社区基层党组织"具体列举
- **不议论**：不写"为什么需要规定任期" "任期重要意义"
- **不铺垫**：不"在 XX 背景下" "为深入贯彻 XX 精神"

## 与 AI 综艺体的对比

❌ **AI 综艺体**：
> 在新时代背景下，党的基层组织建设面临着前所未有的新挑战和新机遇。任期制度作为党内民主的重要体现，对于推动基层党组织建设、提升组织力具有深远意义。本意见旨在通过规范任期制度，进一步加强党的基层组织建设...

✅ **真实公文（本范文风格）**：
> 第一条：基层党委、党总支、党支部，以及村和社区基层党组织的任期，规定如下...

## Transferable Lessons

1. **专项性意见 = 不议论**：直接出规定，不要"重要意义"开头段
2. **综合性意见 = 简短认识**：写认识也要紧扣主题，不能写成论述文
3. **意见 ≠ 论文**：意见的目的是规定行为，不是论证道理
4. **标准末条**：施行日期 + 与其他规定关系
5. **每条规定要可操作**：列举具体对象，不留模糊空间

## 适用场景

- 党内规章意见
- 部门工作意见
- 实施性规定
- 反例：当你想用"在...的指引下，为推动...，特提出如下意见"开头时，回看本范文
