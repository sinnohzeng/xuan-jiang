# 范文 06：述职报告"三取胜"

**来源**：《怎样写作》（任仲然）第九讲「述职报告的写法」
**类型**：述职报告
**学什么**：以实取胜 / 以数取胜 / 以事和绩取胜

---

## 第一胜：以实取胜（力戒华而不实）

### 关键示范

> 本人执行中央八项规定精神的态度是坚决的，不搞任何变通，不得过且过，严格要求自己。带头到社区报到为群众服务，每个季度都带领机关干部深入农村参加集体劳动。**到州市开会或到县乡村检查工作不住宾馆酒店，自带行李吃住在农户了解民情**，帮助基层和群众解决了 50 多个具体问题。

**学什么**：
- "态度坚决"后面立刻跟具体行动（不变通、不得过且过、自带行李吃住农户）
- 不空话"严格要求自己"，而是"50 多个具体问题"具体数字
- 涉及政治立场不说软话，但也不夸张（"态度是坚决的"，不用"绝对""坚决拥护"）

## 第二胜：以数取胜（克服量化不足）

### 关键示范

> 一年来，我先后到 72 个县 240 多个乡镇村庄调研，有的县去过三四次。坚持少开会、开短会、开解决问题的会，**讲真话实话，不讲官话套话**。几次全省电视电话会议只讲十来分钟，直奔主题。所有讲话均自己准备提纲，很少念稿子。**主动控制文件，年内多次讲话只有两次发了内部通报。**

**学什么**：
- **工作量化**：72 个县 / 240 多个乡镇村庄 / 三四次
- **行为量化**：电视电话会议十来分钟、年内只发两次通报
- **节制使用数据**：不每段都堆数字，与定性评价配合（"少开会、开短会"= 定性 + "十来分钟"= 定量）

### 任仲然评点

> 数据是非常有用的要素，量化是特别管用的办法，定性评价的特征需要用文字概括表述，定量评价的特征需要用数据表述。**较好的做法是，定性评价与定量评价相结合，数据和百分比为定性判断服务，由此增强报告的量化感和可信度。**

## 第三胜：以事和绩取胜（用事项和政绩证明）

### 关键示范

> 七、存在的突出问题及改进措施。粮食因灾减产，农民收入增幅较小，制约农业农村发展矛盾凸显；分管部门的自身建设任务较重，机构和队伍不适应新形势需要；综合协调各方支持"三农"的能力有待增强，抓落实力度需进一步加大；下基层接触各级干部多、联系农民群众少，有时也存在走马观花的问题。**针对这些问题，我要加强学习调研、改革创新、协调沟通、队伍建设和自身建设，夯实抓大事、抓重点等基础性和关键性工作，以求真务实的态度解决问题，逐项进行整改。**

**学什么**：
- **不回避问题**：粮食因灾减产、农民收入增幅较小、走马观花
- **问题表述具体**：用分号衔接 4 个具体问题，每个问题都点出具体场景
- **改进措施可落地**：不空话"加强 XX"，而是"逐项进行整改"

## "三胜"使用模板

```markdown
# 一、主要工作（按"事"分类）

## 1. [大事 1]
[具体行动 + 量化数据 + 成效]

## 2. [大事 2]
...

# 二、查找不足（坦诚不回避）

[具体问题 1，分号衔接 + 具体场景]
[具体问题 2，分号衔接]

# 三、改进措施（可落地）

针对这些问题，我要：
[行动 1] + [行动 2] + ...
```

## Transferable Lessons

1. **"实"= 具体行动 + 具体数字**：不空话"严格要求自己"，要"自带行李吃住在农户"
2. **"数"= 工作量 + 成效 + 行为**：72 个县 / 三四次 / 十来分钟 / 两次通报
3. **"事和绩"= 大事讲透 + 小事略写**：浓墨重彩 3-4 件大事，杂事一笔带过
4. **不回避问题**：用分号衔接的具体问题，比"还存在不足"具体得多
5. **改进可落地**：不"加强领导"，要"逐项进行整改"

## 适用场景

- 年度述职报告
- 个人考核述职
- 班子成员述廉述职
- 反例：当你写"成绩属于过去，未来仍需努力"时，回看本范文
