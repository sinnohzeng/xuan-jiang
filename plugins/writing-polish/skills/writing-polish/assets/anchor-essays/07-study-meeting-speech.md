# 范文 07：学习会发言稿

**来源**：《怎样写作》（任仲然）第十讲「汇报稿和发言稿的写法」§ 学习会发言稿
**类型**：学习会发言稿（中心组学习会、专题学习会、读书小组学习会）
**学什么**：三个结合得紧密——平时学习与会上听讲结合 / 理论与实际结合 / 历史纵深与当下现实结合

---

## 写作背景

任仲然在一次中心组学习会上，请知名教授作马克思主义哲学专题辅导。他**一边听教授讲课，一边写学习体会发言稿**，文字粗糙未润色，但**原汁原味**有其价值。

## "三个结合得紧密"特点

### 第一结合：平时学习思考 + 会上听辅导讲座

谈学习体会时**不时引用辅导老师的观点看法**，说的都是经过深入思考的话，但又**完全契合中心组学习的主题**。

**关键点**：不是上来就背老师讲的，也不是只讲自己，而是**两者交织融合**。

### 第二结合：马克思主义哲学理论 + 本省发展实际问题

谈马克思主义哲学可以头头是道，但**不要妨碍联系现实问题的实度和深度**。联系实际谈学习体会，有利于学思践悟，对改进工作能有所推动。

**关键点**：理论谈再深，落脚点还是工作改进。

### 第三结合：历史纵深感 + 理论扎实感 + 当下现实感

在不长的篇幅中，做到了**历史和理论、理论和现实的紧密融汇**。

**关键点**：三个时间维度（历史 / 理论 / 现实）交叉，避免单一维度的扁平表达。

## 任仲然评点

> 另外，**结构上纵横与粗细结合得也比较紧密，语言表达上亦有鲜明特点**。

## "三结合"发言稿模板

```markdown
# [主题] 学习体会

## 一、[第一结合] 听讲启发 + 平时思考

教授今天讲到 XX 时，我突然想起去年读 YY 时的一个问题……

[展开 1-2 段，把老师的观点和自己的思考交织]

## 二、[第二结合] 理论引用 + 实际问题

马克思在 XX 中提出 ZZ。这个判断在我们 [本省 / 本部门 / 本工作] 当下的情况下意味着什么？

[展开具体的本职工作问题]

## 三、[第三结合] 历史 + 理论 + 现实

回顾历史，[历史维度]；理论上，[理论维度]；当下，[现实维度]。三者形成了 [一个具体判断]。

[最后落地到 1-2 个具体行动]
```

## 解析

- **立意**：学习会发言稿的灵魂是"结合"——纯粹谈理论是论文，纯粹谈工作是工作汇报，**学习会发言稿是两者的化合物**
- **结构特点**：纵横（时间 + 空间）粗细（理论深度 + 工作细节）结合
- **语言特点**：
  - 老师课上观点不要直接引用大段
  - 自己思考用第一人称（"我突然想起""在我们这里"）
  - 不堆砌哲学术语
- **可摹仿点**：
  - 写作时机选择"边听边写"——保留新鲜感
  - 不刻意润色——**原汁原味**比技术化加工更可贵

## Transferable Lessons

1. **学习发言稿不是论文**：理论必须落地到本职工作
2. **结合不是 50/50 分配**：是交织融合，理论一句、感受两句、问题三句、回到理论一句
3. **保留思考过程**：不强行精修，让读者看到你思考的真实路径
4. **用第一人称**："我突然想起""在我们这里"——避免"专家学者认为"模糊归因
5. **历史/理论/现实三维度交叉**：避免单维度抒情或抽象议论

## 适用场景

- 中心组学习会发言
- 专题理论学习
- 读书心得交流
- 反例：当你想直接引用 5 段老师的观点 + 抒发"我深受启发"时，回看本范文
