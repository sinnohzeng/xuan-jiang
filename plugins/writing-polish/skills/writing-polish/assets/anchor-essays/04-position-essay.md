# 范文 04：问题是个好东西

**来源**：《怎样写作》（任仲然）第三讲第一节「立意要高深」
**类型**：议论文 / 短论
**学什么**：立意贯穿——从一句名言到一个鲜明主题的转化过程

---

## 写作背景

老石在一个会议上听到"问题就是时代的口号"这个经典表述，**瞬间触动了写作敏感**。会议期间写下短文发表，后来据此组织拍摄基层组织建设问题视频短片，再后写下面这篇文章。

## 原文（节录）

> "问题就是时代的口号"，这是多么博大深邃的一句话啊！刚听到这句话的时候，我立即就意识到了马克思的透彻洞察和辽远视野。
>
> ……
>
> 在思想巨匠面前，我的语言没有办法不贫乏，循着伟人的思维去认识问题，总能感受到自己的落差。此时只好直白地写出自己的感慨，"问题"在我这里便由压力、挑战、困难变成了时代的声音，成了"好东西"。本人为此写了一篇关于问题是个好东西的短文，表达的不是抽象的学习心得，而是自己的实践体验。
>
> ……
>
> 回顾把问题当成好东西的实践过程，可以看到一个很有价值的思维路径及思想脉络，这也是对马克思关于问题是时代的口号这一绝妙表述的诠释——时代的每一点滴进步，问题总是走在它前头。**发现了问题，就发现了时代变化的"沟"在哪里，"坎"在哪里**；研究了问题，就研究了时代发展的方向及道路的选择；解决了问题，就推动了时代的车轮滚滚向前。

## 解析

- **立意**：把马克思一句话转化为工作宗旨——**好立意是好文章的基本前提**
- **从立意到落地的全链路**：
  1. 听到名言 → 感觉触动（捕捉灵感）
  2. 写小短文 → 发表
  3. 拍视频短片 → 在多个会议播放（产品化）
  4. 再写长文 → 系统呈现立意贯穿过程
- **语言**：用具体动作分解抽象概念——"发现 → 研究 → 解决"对应"沟在哪 → 方向选择 → 推动车轮"
- **可摹仿点**：
  - 立意来源于真实触动（具体场景：在会议上听到这句话）
  - 不抽象议论，而是结合"分管的工作实际"
  - 把"好东西"作为反传统判断（问题通常被视为压力，但作者把它定义为"好东西"）

## Transferable Lessons

1. **立意来自触动而非检索**：好立意不是想出来的，是被某句话/某件事触发后顺势抓住的
2. **从立意到产品的多介质转化**：短文 → 视频 → 长文——同一个立意可承载多种产出形式
3. **反传统判断的力量**：把"压力/挑战"重定义为"好东西"，比"我们要重视问题"有冲击力
4. **不抽象议论**：用"发现/研究/解决"三动作 + "沟/坎/方向/车轮" 具象比喻

## 适用场景

- 工作随笔、心得体会、专题论文
- 需要"立意鲜明"的任何文章
- 反例：当你想用"我们要充分认识 XX 重要意义"开头时，回看本范文
