# 范文 03：生态兴省高层论坛演讲稿

**来源**：《怎样写作》（任仲然）第四讲 § 老石 2009 年论坛演讲稿
**类型**：演讲稿
**学什么**：结构紧凑严实——五点感想，层级适中

---

## 原文（节录）

> **尊敬的各位专家，各位朋友，各位同志：**
>
> 大家上午好！
>
> 我省"绿色转型、林业崛起、生态兴省"论坛，今天在这里隆重举办。著名专家学者，各方有识之士，将奉献一坛生态美景。这次论坛的定位是，生态战略之坛，林业振兴之坛，呼唤绿色、迎接春天、开辟美好未来之坛。我们要通过这次论坛，形成高度的生态兴省共识，营造强大的绿色转型态势。我讲五点感想，可能占用大家七分钟时间。
>
> **第一，我讲一讲去年的两个会议。** 一是全国造林绿化现场会在我省召开，盛况空前，成效卓著，各省反响强烈，我省荣获全国造林绿化唯一的突出贡献奖。二是省委书记和省长在省委林业工作会议上，提出了实施生态兴省战略的愿景构想。这两个会议，标志着我省林业步入了快速发展、全面提升的新轨道，进入了绿色转型、生态兴省的新时期。
>
> **第二，我想讲一个重要关系。** 今天，我可以什么都不讲，但是我一定要讲一讲林业和生态的关系。这个关系简要地说，就是没有林业就没有生态。许多地方不搞植树造林，生态建设就是一句空话。林业是生态之本，生态之基，生态之母。那么，生态是什么呢？生态是林业最亲的一个儿子。
>
> **第三，我讲两个故事。** ……一位农民提出个问题，他问我碳汇造林怎样才能与退耕还林结合起来？……一位省军区的师职干部马上就要退休了，他说自己什么都不想干，就想回老家去种树。这两个故事说明，我省林业生态建设的社会基础是多么深厚，干部群众发展林业的愿望是多么强烈。
>
> **第四，介绍一下林业生态省建设的阶段性目标。** 从今年开始，要用十年的时间，初步建成林业生态省。大体分两个阶段，五年实现绿色转型，十年实现生态兴省。我提交的论文中，初步设计了十项指标，仅供大家参考。
>
> **第五，还要说说我自己。** ……我要在这里落地归根——要像国槐、杨柳那样，深深扎根在母亲河两岸，要像柠条、沙棘那样，执着生长在黄土高原。
>
> 就我的年龄来说，还可以在省政府工作五年。未来的五年，其他事情可能会少干一些，但林业生态省建设，我要竭尽全力干到底。
>
> 谢谢同志们！

## 解析

- **立意**：以个人感想方式表达对生态省建设的真诚态度和坚定决心，**避免大而不当的结构惯性**
- **结构**：五点感想，每点一句话标题——"讲两个会议 / 讲一个关系 / 讲两个故事 / 讲阶段性目标 / 讲我自己"，层级适中、不过多不显少
- **语言**：
  - 第二点用"林业是生态之本，生态之基，生态之母" + "生态是林业最亲的一个儿子"——把抽象关系讲得通俗
  - 第三点用具体故事（农民提碳汇问题、军人要退休种树）取代理论说教
  - 第五点结尾用比喻"像国槐、杨柳那样深深扎根""像柠条、沙棘那样执着生长"——情真意切
  - 结尾克制"未来的五年……竭尽全力干到底"

## Transferable Lessons

1. **七分钟讲完五点**：不堆砌大量内容，时间和层级控制是结构的灵魂
2. **每点标题用"我讲...、我想讲...、还要说说..."第一人称**：不用"关于 XX 的若干思考"AI 化结构
3. **理论问题用具体故事讲**：碳汇造林 + 退耕还林 = 抽象政策；农民问"怎样结合" = 具体故事
4. **抒情结尾用比喻**：避免"贡献力量"等空话，用"像国槐、杨柳那样"等具象
5. **不搞三大块、四大块套路**：五点感想就是五点，不刻意凑成三段

## 适用场景

- 论坛演讲、专题发言
- 需要表达个人态度的讲话
- 反例：当你想搞"理论篇 / 实践篇 / 展望篇"机械三段时，回看本范文
