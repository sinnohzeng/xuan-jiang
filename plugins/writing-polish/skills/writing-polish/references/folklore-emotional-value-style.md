# 民俗玄学与情绪价值类长文写法

适用于用户明确要求“只聊玄学 / 民俗 / 情绪价值，不聊医学或科学判断”的中文长文。

## 结构

1. 先写“整体讲述”：部位、民俗说法、去留原则、情绪价值框架。
2. 再写“具体对象分析”：只在最后单独分析用户给出的个人情况。
3. 不要在整体讲述里反复代入个人名字；整体部分写成通用说明，个人部分再点名。
4. 需要飞书交付时，长篇优先写入飞书云文档，不要在聊天中展开全文。

## 口吻

- 克制、温柔、少断言，多用“可以理解为”“更容易被说成”“民俗里常讲”。
- 明确这是文化叙事和情绪价值，不作事实预测。
- 把决策说成“整理”“保留”“留一点故事”，不要写成“必须”“一定”。

## 作家风格锚定

可锚定汪曾祺散文方向：清淡、家常、温和，像饭桌边慢慢讲一件小事。只借气质，不摹仿具体句子。

## 痣相类可复用框架

- 总原则：显处去杂，隐处留福。
- 显处：脸、额头、鼻翼、嘴边、脖子前面、小臂外侧、手背。审美优先，可说成“清面”“净气”“整理外缘”。
- 隐处：眉毛里、发里、后颈、腋下、屁股、脚底、身体背面。更适合说成“藏福”“暗财”“贵人”“私库”。
- 半显半隐：大臂、小臂内侧、大腿外侧、脚背、前胸。以本人照镜子是否舒服为准。
