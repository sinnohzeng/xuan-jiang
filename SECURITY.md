# 安全策略 Security Policy

## 范围 Scope

本策略仅覆盖轩匠仓库中的 writing-polish 插件代码。Claude Code 本身的安全问题请向 [Anthropic](https://docs.anthropic.com/en/docs/claude-code/security) 报告。

This policy covers only the writing-polish plugin code in this repository. For Claude Code itself, report to [Anthropic](https://docs.anthropic.com/en/docs/claude-code/security).

## 报告漏洞 Reporting a Vulnerability

如果你发现安全漏洞，请通过以下方式报告：

1. **GitHub 私有安全报告**（推荐）：前往 [Security Advisories](https://github.com/sinnohzeng/xuan-jiang/security/advisories/new) 创建私有报告
2. **GitHub Issue**：对于非敏感的安全改进建议，可直接创建 Issue

If you discover a vulnerability:

1. **GitHub Private Security Advisory** (preferred): Create a private report at [Security Advisories](https://github.com/sinnohzeng/xuan-jiang/security/advisories/new)
2. **GitHub Issue**: For non-sensitive security improvements, open an Issue

## 处理流程 Response Process

1. **确认**：收到报告后 7 日内确认
2. **评估**：确定影响范围和严重程度
3. **修复**：发布修复版本并在 commit 中说明

## 支持的版本 Supported Versions

仅支持各插件的最新版本。请确保使用最新版本后再报告问题。

Only the latest version of each plugin is supported. Please ensure you are using the latest version before reporting.
