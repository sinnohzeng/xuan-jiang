## v5.1 vs v5.0 关键演进

### 偏离 1 修复：placeholder 模板 → 主智能体判断指南

v5.0 `prompts/multi-agent/` 5 个模板带 `{{VIEW_ID}}` `{{ROLE_TITLE}}` `{{ABSOLUTE_FILE_PATH}}` `{{GENRE}}` `{{MAX_FINDINGS}}` 等 7+ 占位符，主对话需手工 copy-paste + fill placeholder + 调 Agent 工具——**这是机械填空，不是智能判断**。

v5.1 砍成单一 `orchestration-guide.md` 10 段：
1. 何时启 L3 / 2. 派几个 reviewer / 3. 视角组合 / 4. Agent 工具具体语法 / 5. Pre-mod 触发 / 6. R2 fresh-eye 触发 / 7. 收敛判停 P0-P5 + 决策三问 / 8. 模型路由 1 Opus + N Sonnet / 9. 错误恢复 fallback / 10. Context Budget 自检 / 11. jsonl 记录

主对话**根据文稿特征自主判断**而非填模板。Anthropic Multi-Agent System 2025-06 "teach orchestrator how to delegate" 范式落地。

### 偏离 2 修复：D5/D4 few-shot 不足 → 8 个新 example + 2 个雷达

v5.0-rc1 calibration κ=0.368，D5 / D4 κ=0（D5 accuracy 74.5% 但 judge 总输出"无违规"被 prompt 固化）。

v5.1 从 calibration-results-baseline-v50rc1/disagreement.md 18 个 disagreement 直接提炼：
- Example G-K（5 个 D5 反例）：充分 + 大幅 + 不再仅仅是 / 不仅 X 更是 Y / 有必要首先 / 经济学抽象化 / 随着 X 的深入 ... 演变为 ...
- Example L+M（2 个 D4 反例）：复盘在 G8 违规 / 复盘跨文体对照（G2 大厂 vs G8 咨询）
- Example N（1 个 D1 反例）：半角括号紧跟英文术语

constitution.md §6.1 G3/G8 D5 模糊副词专属雷达（4 类信号清单）+ §6.2 D4 同词跨文体条件判决表（6 词 × 4 文体 = 24 cell 矩阵）。

constitution.md §2.8 §1.8.6 大厂内训词侵入红线（v5.1 新增）："复盘 / 拉通 / 对齐颗粒度 / 跑通 / 收口" 在 G8 咨询语境出现 → D4 ≥ 2。

### 偏离 3 修复：SKILL.md 344 行 → 90 行（Anthropic Progressive Disclosure）

外迁 6 大段到 references/：DOCX 决策树 → docx-editing-guide.md / 写作 5 步法 → writing-methodology.md / 审稿 3 步法 → revision-checklist.md / L1 三小节 → anti-ai-taste-anchors.md / L2 步骤 → constitution.md / L3 步骤 → orchestration-guide.md / 修改哲学 → revision-checklist.md。
